# Dagbók

Ég bjó til GOGN möppu og í henni möppurnar Sameign, Forritun og Markadmal.

**GOGN** mappan:
cd /
sudo mkdir GOGN
cd
ls -l / (gá hvort GOGN sé í root)
cd /
sudo chown ubuntu GOGN
ls -l  (gá hvort það virkaði að breita root í ubuntu)

Hinnar möppurnar:
cd GOGN
mkdir Sameign
ls  (gá hvort það virkaði)
mkdir Forritun
ls
mkdir Markadsmal
ls
